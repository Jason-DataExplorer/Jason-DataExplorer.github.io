TAIWAN找茶官網上線啦！

By Friends from NTUB 12 August 2021

關於TAIWAN找茶

柴米油鹽醬醋茶是古今「開門七件事」，但你上次是什麼時候喝茶呢？

台灣茶的好舉世皆知，但若真要深究，大多數人其實也一知半解。

我們希望能讓國人更認識生長於自己土地上的茶葉，而「TAIWAN找茶-B2B2C電商平台」則為此誕生。

Dimension Description
Est. 2021.01.01
Core Value Authentic Tea
Origin Taiwan

TAIWAN找茶期許能成為台灣茶之電商整合平台，

在地茶農&茶行除可於平台上「開店」販售茶葉外，更可利用本平台特色-找茶Chatbot，

使消費者輕鬆獲得從茶葉分類、沏茶方式，乃至食安議題等與茶葉有關的所有資訊。

然而茶葉的品項琳瑯滿目。因此 「如何選擇？」 也變得尤為重要，

找茶Chatbot透過大數據分析，提供客製化推薦並與本平台連結，

望能拉近消費者與台灣茶農&茶行的距離、協助消費者得到更好的服務品質，

進而提升台灣茶產業的環境。茲整理本企劃之核心目的如下：

  1. 建立台灣茶之整合平台，使消費者能與在地茶農&茶行更加靠近。

  2. 藉由將茶葉資訊導入Chat bot，使消費者能輕鬆獲取更透明的生產者資訊。

  3. 促進在地茶農&茶行的科技轉型，攜手推廣茶葉文化、建立產業的良好循環。
